# TODOカレンダー

AIが入力内容からタスクの重要度を自動判定する、TODOカレンダーWebアプリケーションです。Python (Flask) と HTML/CSS/Vanilla JS を使用して構築されています。

## 特徴

- **AIによる重要度判定**: タスクのタイトルや詳細文に含まれるキーワードから、重要度（高 / 中 / 低）を自動で判定します。
- **シンプルな操作性**: カレンダーから日付を選択し、タスクの追加・完了・削除を直感的に行えます。

## 使用技術
- **フロントエンド**: HTML5, Vanilla CSS, Vanilla JavaScript
- **バックエンド**: Python, Flask
- **データベース**: SQLite3 (標準ライブラリ)

## セットアップと起動方法

### 前提条件
- Python 3.x がインストールされていること

### 手順

1. **仮想環境の作成と有効化**
   ```bash
   # 仮想環境の作成
   python3 -m venv venv

   # 仮想環境の有効化 (macOS / Linux)
   source venv/bin/activate
   # Windows の場合: venv\Scripts\activate
   ```

2. **Flaskのインストール**
   ```bash
   pip install flask
   ```

3. **アプリケーションの起動**
   ```bash
   python app.py
   ```

4. **ブラウザでアクセス**
   起動後、ブラウザを開き `http://localhost:8080` にアクセスしてください。

## AI判定機能の拡張について
現在のAI判定ロジックは `app.py` 内の `evaluate_importance_ai` 関数で実装されているモック（キーワードベースの判定で代用）です。
この部分を、Google Gemini API などの大規模言語モデル(LLM)のAPI呼び出しに置き換えることで、文脈を理解したより高度な判定機能にアップグレードさせる予定です。
